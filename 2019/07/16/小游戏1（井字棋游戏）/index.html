小游戏1（井字棋游戏）

Posted on 2019-07-16

游戏简介

三连棋游戏(两人轮流在印有九格方盘上划“+”或“O”字, 谁先把三个同一记号排成横线、直线、斜线, 即是胜者)。
程序提供随机算法和智能算法两种AI，随机算法使用随机数随意选择棋盘上的位置，智能算法通过对每隔落子位置权重的计算，选取最优的落子点。

游戏实现

程序解析：

  • 程序主要有Competition和Competitor组成。RandomAI、IntermediateAI和Human都属于Competitor，每个类都封装了下棋函数，每次把棋子和棋盘情况传递给下棋函数，改变棋盘的记录。Competition中进行对棋手的选择、胜负的判断和棋盘情况的打印。

  • 智能的算法是在每一步前对当前位置的权值进行判断，比如AI使用O，那么两个O的位置就是优先下的位置，两个X的位置就是次级优先的，并且权值远远大于一个X的位置；一个X的位置又稍大于一个O的位置，并大于空行。所以O2 = 10000,X2 = 1000, O = 10, X = 6,Empty = 4。再计算完成每一行、每一列以及斜列的权值总和最高的空置就是落子的最佳点。

    测试

  • 使用catch作为测试工具进行单元测试。

  • 两个AI的对战测试程序，智能AI比随机AI高80%的胜率。
